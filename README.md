## 🇬🇧 English

A powerful Python tool specifically designed for compiler developers to convert LLVM SelectionDAG Graphviz `.dot` files into yEd-compatible `.graphml` files.

When using LLVM's `llc` tool to generate DAG dot files, directly viewing them as images lacks editability. Existing converters often fail to parse LLVM's complex `Mrecord` formats and lose edge styles. This script solves that by structuring the node labels into beautiful HTML tables and preserving all layout and styling information for seamless drag-and-drop editing in [yEd Graph Editor](https://www.yworks.com/products/yed).

### ✨ Features
*   **Intelligent Record Parsing**: Automatically parses LLVM's messy `{<s0>0...}` strings into structured fields: Inputs, Operator, Node ID, and Outputs.
*   **Native HTML Tables**: Renders nodes as native HTML tables in yEd, perfectly matching the original Graphviz visual structure.
*   **Preserves Layout & Styles**: Extracts node coordinates from Graphviz and scales them for better readability. Fully restores LLVM edge styles (e.g., `blue/dashed` for control flow, `red/bold` for Glue/Chain).
*   **Auto Size Policy**: Nodes dynamically resize to wrap their internal text perfectly.

### 🛠 Prerequisites
*   Python 3.x
*   [Graphviz](https://graphviz.org/) installed and `dot` added to your system's PATH.

### 🚀 Usage

```bash
python3 dot2xml.py input.dot output.graphml
```

1. Generate your LLVM DAG dot file (e.g., via `llc -view-isel-dags`).
2. Run the script.
3. Open `output.graphml` in yEd.

### 📂 Examples
Check out the `/examples` directory for a sample `main.dot` generated by LLVM and the resulting `main.graphml`.

## 🇨🇳 中文说明

专为编译器开发者打造的 Python 工具，用于将 LLVM SelectionDAG 生成的 Graphviz `.dot` 文件转换为 yEd 可编辑的 `.graphml` 文件。

在使用 LLVM 的 `llc` 工具生成 DAG 图时，直接导出为图片无法进行二次拖拽编辑。现有的转换工具往往无法正确解析 LLVM 复杂的 `Mrecord` 格式，且会丢失依赖边（如 Glue、Chain）的颜色和样式。本工具通过 AST 级别的解析，将节点标签重构为精美的 HTML 表格，并完美保留所有布局坐标与连线样式，方便在 [yEd Graph Editor](https://www.yworks.com/products/yed) 中进行深入分析与排版。

### ✨ 核心特性
*   **智能 Record 解析**：自动将 LLVM 杂乱的 `{<s0>0...}` 字符串解构为：输入 (Inputs)、操作符 (Operator)、节点 ID 和输出 (Outputs)。
*   **原生 HTML 表格渲染**：在 yEd 中使用原生 HTML Table 还原节点，完美重现 Graphviz 中的表格视觉结构。
*   **保留布局与样式**：自动提取 Graphviz 的布局坐标并进行智能缩放防重叠；100% 还原 LLVM 连线样式（例如蓝虚线表示控制流，红粗线表示 Glue）。
*   **自适应节点大小**：文本节点注入自适应策略，完美包裹内部表格，拒绝文字溢出。

### 🛠 环境依赖
*   Python 3.x
*   安装 [Graphviz](https://graphviz.org/) 确保 `dot` 命令已加入系统环境变量 PATH 中。

### 🚀 使用方法

```bash
python3 dot2xml.py input.dot output.graphml
```

1. 使用 LLVM 生成你的 DAG dot 文件（例如通过 `llc -view-isel-dags`）。
2. 运行上述转换脚本。
3. 使用 yEd 软件打开生成的 `output.graphml` 文件即可自由拖拽编辑。

### 📂 示例 (Examples)
你可以进入 `/examples` 目录，查看 LLVM 生成的原始 `main.dot` 文件以及通过脚本转换后精美的 `main.graphml` 文件。
